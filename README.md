# gpt-image-2 山海经实验仓库

这个仓库现在按“未来部署到 Vercel 的山海经图鉴站点”来组织内容。

## 目录结构

- `content/creatures/`
  - 每个异兽一个独立目录，保存结构化元数据、提示词、后续文案和展示配置。
- `public/images/creatures/`
  - 网站直接引用的最终图片资产。这里的文件路径可以稳定映射到未来站点 URL。
- `data/catalogs/`
  - 从外部资料抽取、清洗后的总表数据，用于批量生成页面或内容索引。
- `scripts/`
  - 数据抓取与构建脚本。
- `src/`
  - 预留给未来 Vercel 站点代码。建议后续采用 Next.js App Router。
- `docs/`
  - 项目结构、内容规范、后续建站约定。
